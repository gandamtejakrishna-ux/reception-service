# Reception Service

This service exposes APIs for booking, check-in, check-out, room availability, booking details etc.,.

## Port
Runs on: **http://localhost:9001**

## Prerequisites
- Java 17+
- sbt
- Booking Service running on port **9002**
- JWT token required for all booking APIs

## Start the Service
sbt run
